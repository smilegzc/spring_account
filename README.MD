# 基于SpringBoot和MyBatis的商店销售帐目和库存管理系统(近期不准备更新，以后更新随缘)

## 开发环境
  - IDE: Intellij IDEA 2019.3.4
  - JDK：Oracle JDK 14
  - MySql: 8.0.19
  - maven: 3.6.3
  - Git: 2.26.0
## 运行依赖
  - 请查看pom.xml文件
## 使用方法
  - 修改yml配置文件数据库配置信息,按照resources/SQL目录下的sql文件创建数据表
  - 直接导入IDEA使用
  - 下载release文件夹下的jar包直接使用java命令运行
